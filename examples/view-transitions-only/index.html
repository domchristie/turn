Turn: View Transitions

View Transition

This demonstrates animated page navigations with View Transitions and Turn. Tap the Turn/Lorem links in the navigation bar to view. No custom CSS is required for the default animations, but View Transitions need to be enabled in Turn:

Turn.config.experimental.viewTransitions = true

Your browser does not support View Transitions, so you'll not see any animations.
